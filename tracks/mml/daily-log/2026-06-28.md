Completed Lesson:
28. PCA

Key Insight:
PCA finds the most important directions in data. It reduces many features into fewer new features while keeping as much variance as possible.

Concrete Example:
If points on a 2D chart mostly stretch along one diagonal line, PCA finds that diagonal as the first principal component. The data can then be described mainly by positions along that direction.

Main Formula Ideas:

Center the data:

\[
x_i' = x_i - \bar{x}
\]

Build the covariance matrix:

\[
\Sigma = \frac{1}{n}X^T X
\]

Solve the eigenvalue equation:

\[
\Sigma v = \lambda v
\]

ML Connection:
PCA is used for dimensionality reduction, visualization, compression, noise reduction, and reducing feature redundancy.
