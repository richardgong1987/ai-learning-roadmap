28. PCA

Key Insight:
PCA finds the most important directions in data. It reduces many features into fewer new features while keeping as much variance as possible.

Concrete Example:
If points on a 2D chart mostly stretch along one diagonal line, PCA finds that diagonal as the first principal component. The data can then be described mainly by positions along that direction.

Main Formula Ideas:

Center the data:

\[
x_i' = x_i - \bar{x}
\]

Build the covariance matrix:

\[
\Sigma = \frac{1}{n}X^T X
\]

Solve the eigenvalue equation:

\[
\Sigma v = \lambda v
\]

ML Connection:
PCA is used for dimensionality reduction, visualization, compression, noise reduction, and reducing feature redundancy.

29. Gradient Descent

Key Insight:
Gradient descent reduces loss step by step by moving parameters in the opposite direction of the gradient.

Concrete Example:
For L(w) = (w - 3)^2, starting at w = 0 with learning rate 0.1 gives gradient -6 and update w_new = 0 - 0.1(-6) = 0.6. The parameter moves toward the minimum at w = 3.

Main Formula:

\[
\theta_{new} = \theta_{old} - \alpha \nabla L(\theta_{old})
\]

ML Connection:
Gradient descent trains machine learning models by repeatedly updating weights and biases to reduce loss.
