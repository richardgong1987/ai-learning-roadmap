Completed Lesson:
19. Taylor Series

Key Insight:
A difficult function can be approximated near a point by using value, slope, curvature, and higher-order derivative information.

Concrete Example:
e^x near x = 0 can be approximated by 1 + x + x^2/2! + x^3/3! + ...

ML Connection:
Taylor Series supports local approximation, gradient descent intuition, and second-order optimization methods.
