27. Linear Regression

Key Insight:
Linear regression learns a straight-line relationship between input features and a numeric target. It chooses weight and bias values so predictions are close to real values.

Concrete Example:
For study hours and exam scores, the line y_hat = 10x + 40 predicts 50 for 1 hour, 60 for 2 hours, and 70 for 3 hours.

Main Formula:

\[
\hat{y} = wx + b
\]

Loss Function:

\[
L(w,b) = \frac{1}{n}\sum_{i=1}^{n}(wx_i + b - y_i)^2
\]

ML Connection:
Linear regression is a supervised learning model. It connects functions, gradients, loss minimization, covariance, and prediction.
