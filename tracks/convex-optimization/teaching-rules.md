# Convex Optimization Teaching Rules

These rules are the source of truth for teaching Richard Convex Optimization.

## Language

- Teach the main lesson in English.
- Always include key technical terms in English plus Chinese.

## Key Technical Terms Rule

Each lesson must include a section called:

`Key technical terms (English + Chinese)`

Example:

- Convex Optimization: 凸优化
- Convex Set: 凸集
- Convex Function: 凸函数
- Objective Function: 目标函数
- Constraint: 约束
- Duality: 对偶性
- KKT Conditions: KKT 条件

## Teaching Style

- Explain like teaching a primary-school student.
- Start with a simple real-life example.
- Use concrete numbers before abstract formulas.
- Do not jump directly into symbols.
- Explain slowly from simple to difficult.

## Formula Rule

For every important formula:

1. Explain what problem the formula solves.
2. Explain every symbol.
3. Plug in actual numbers.
4. Calculate step by step.
5. Then explain the abstract form.
6. Explain where it is used in machine learning, finance, engineering, or optimization.

## Required Lesson Structure

Each lesson should include:

1. Simple real-life problem
2. Concrete numerical example
3. Key technical terms (English + Chinese)
4. Symbol explanation
5. Formula explanation
6. Step-by-step calculation
7. Connection to applications
8. Practical tips
9. 2-3 tiny practice questions
10. Answers and explanations

## Completion Rule

A lesson is completed only after the GitHub repository has been updated.
