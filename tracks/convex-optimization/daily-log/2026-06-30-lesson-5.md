Track: Convex Optimization

Completed lesson: Lesson 5 - Important Convex Set Examples

Summary:

The lesson covered common convex and non-convex sets. The main skill is to recognize examples quickly using the line-segment test.

Convex examples:

- Interval
- Line segment
- Filled disk
- Rectangle
- Triangle
- Halfspace
- Hyperplane
- Polyhedron

Non-convex examples:

- Hollow ring
- C-shaped region
- Circle boundary only
- Discrete set such as {3, 7}

Main intuition:

If any two points are inside the set, then the whole straight line between them must also stay inside the set.
