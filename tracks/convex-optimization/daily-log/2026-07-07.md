## What was learned

### Lesson 8

- Convex functions can be built from smaller convex functions using preservation rules.
- Nonnegative weighted sums preserve convexity.
- Affine composition preserves convexity.
- Pointwise maximum preserves convexity.
- Composition rules require checking convexity and monotonicity conditions.

### Lesson 9

- A conjugate function measures the largest gap between a linear function and the original function.
- The Fenchel conjugate is defined as f*(y) = sup_x (y^T x - f(x)).
- For f(x) = x^2, the conjugate is f*(y) = y^2 / 4.
- Conjugate functions are a bridge to duality, support functions, and Lagrange dual problems.

### Lesson 9

- f(x) = x^2
- f*(4) = max_x (4x - x^2) = 4
- General result: if f(x) = x^2, then f*(y) = y^2 / 4

## Practical takeaway

Before proving convexity from scratch, first try to recognize whether the function is constructed from known convex building blocks using valid preservation operations.

For conjugate functions, remember the simple meaning first: f*(y) is the biggest gap between the line yx and the curve f(x). This idea prepares the next topics: convex optimization problems and duality.
