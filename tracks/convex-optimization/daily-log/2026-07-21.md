## Numerical result reviewed

For the bakery problem:

\[
\begin{aligned}
\text{maximize}\quad &300x+500y\\
\text{subject to}\quad &2x+y\leq8\\
&x+2y\leq8\\
&x,y\geq0
\end{aligned}
\]

The continuous optimum is:

\[
(x^*,y^*)=\left(\frac83,\frac83\right)
\]

with optimal value:

\[
p^*=\frac{6400}{3}\approx2133.33
\]

## Main takeaway

A linear optimization problem finds the best value of a linear objective while obeying linear rules. Its feasible set is convex, so a local optimum is globally optimal; when a finite optimum exists, at least one optimum can be found at a vertex of the feasible polyhedron.
