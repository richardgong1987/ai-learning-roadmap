## Key ideas learned

### Lesson 1

- Mathematical optimization means choosing the best possible decision from many possible choices.
- An optimization problem usually contains decision variables, an objective function, constraints, a feasible set, and an optimal solution.
- A feasible solution must obey all constraints.
- An optimal solution is the best feasible solution according to the objective.

### Lesson 2

- Least squares means choosing the best approximate fit when exact matching is impossible.
- A residual is the difference between a prediction and the real observed value.
- Squared error prevents positive and negative errors from cancelling each other.
- Linear programming means optimizing a linear objective under linear constraints.
- In linear programming, the best solution often occurs at a corner of the feasible region.

### Lesson 3

- Convex optimization is powerful because it avoids hidden local traps.
- A convex set has the property that the straight line between any two allowed points stays inside the set.
- A convex function bends like a bowl for minimization.
- In convex minimization, any local minimum is also a global minimum.
- Convexity makes optimization results easier to trust in machine learning, finance, engineering, and solver practice.
