Completed lesson: Lesson 7 - Convex Functions

Summary:

The lesson covered the definition and intuition of convex functions. The main idea is that a convex function stays below the straight line connecting any two points on its graph.

Key formula:

f(theta x + (1 - theta)y) <= theta f(x) + (1 - theta)f(y)

Main example:

For f(x) = x^2, x = 0, y = 2, and theta = 0.5:

- Mixed input = 1
- f(1) = 1
- Weighted endpoint value = 2
- Since 1 <= 2, the example satisfies convexity

Main intuition:

A convex function is like a bowl or valley. For minimization, this is useful because local minima are also global minima.

Applications discussed:

- Machine learning loss functions
- Finance risk minimization
- Engineering cost or energy functions
- Optimization objective functions
