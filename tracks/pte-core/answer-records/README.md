# PTE Core Answer Records

This folder stores Richard's real PTE Core writing attempts.

Each completed writing practice should create one record.

Naming rule:

`YYYY-MM-DD-task-type-topic.md`

Each record should include:

1. Task type
2. Original question
3. Exam instructions
4. Richard's answer
5. Scoring rubric
6. Estimated score
7. Mistake analysis
8. Corrected version
9. Model answer
10. Sentence patterns to remember
11. Next action
